# Edu Admin

`Edu Admin` 是一个面向中小培训机构的轻量运营后台起步项目。

当前这版已经不只是工程骨架，还补了一套可演示的数据和页面，方便继续往真正业务系统推进。

## 当前包含

- Go 后端入口和模块化目录
- Gin 路由骨架
- Vue 3 后台前端壳子
- 登录页与基础权限拦截
- 首页概况、老师、学员、班级、排课、通知演示页
- MySQL 首版建表草案
- 项目规划文档

## 当前可演示内容

- 登录后进入后台首页
- 查看首页概况卡片、近期课表、最近通知
- 查看老师列表
- 查看学员列表
- 查看班级列表
- 查看排课列表
- 查看通知列表

默认演示账号：

- 账号：`admin`
- 密码：当前演示版不做真实校验，前后端跑起来后可直接登录

## 目录说明

- `cmd/server`: 服务入口
- `internal/app`: 启动、配置、中间件、路由
- `internal/modules`: 业务模块
- `internal/platform`: 数据库、日志、权限等基础能力
- `web/admin`: 后台前端工程
- `migrations`: 数据库迁移草案
- `docs/planning`: 项目前期规划文档

## 本地启动建议

1. 复制 `.env.example` 为 `.env`
2. 当前演示版可以先不连 MySQL，后端数据先走内置演示内容
3. 执行 `go mod tidy`
4. 执行 `go run ./cmd/server`
5. 前端进入 `web/admin` 后执行 `npm install`
6. 执行 `npm run dev`

## 当前状态

这还是第一版起步项目，当前主要目标是：

- 让目录、模块和接口边界先稳定下来
- 先把最核心的行业后台页面跑通
- 让后续真正写业务时不需要再推翻基础结构
