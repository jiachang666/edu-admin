# Edu Admin

一个面向学校、培训机构与教培团队的开源后台项目。

`Edu Admin` 想做的不是通用后台壳子，而是一套更贴近真实教培场景的业务后台起步方案，先把课程、学员、班级、排课、签到、作业、通知这些高频环节串起来，再逐步补齐权限、财务、续费、数据统计等能力。

## 项目定位

- 面向中小型培训机构、校区团队、教务运营团队
- 面向想快速搭建行业后台的开发者
- 参考通用后台项目的工程组织方式，但重点放在教培行业的真实业务流程

## 当前状态

当前版本是 `v0.1` 起步版，已经具备：

- 后端基础工程结构
- 前端后台基础布局
- 登录页与基础访问拦截
- 本地 MySQL 自动建库、建表与首批数据初始化
- 首页概况演示页
- 老师、学员、班级、排课、通知演示页
- 首版数据库草案

这一版的目标不是一次把业务做全，而是先把项目骨架、页面方向和模块边界定稳。

## 本地启动

### 1. 启动后端

```bash
git clone https://github.com/jiachang666/edu-admin.git
cd edu-admin
cp .env.example .env
go run ./cmd/server
```

默认后端地址：

- `http://localhost:8080`

默认本地数据库配置：

- 地址：`127.0.0.1`
- 端口：`3307`
- 账号：`root`
- 密码：`123456`
- 数据库：`edu-admin`

### 2. 启动前端

```bash
cd web/admin
npm install
npm run dev
```

默认前端地址：

- `http://localhost:5173`

### 3. 登录演示

- 账号：`admin`
- 密码：当前演示版不做真实校验，填写任意内容即可进入

```text

├── internal/app             # 启动、配置、路由、中间件
├── internal/modules         # 业务模块
├── internal/platform        # 数据库、日志、权限等基础能力
```
